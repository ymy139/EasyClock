## EasyClock
一个简单的学习时钟
第二十五届郑州市学生信息素养提升实践活动-计算思维类-创意编程-初中组 参赛作品

### 功能介绍/开发计划
 - [ ] 基本功能
   - [ ] 时间/日期/节日显示
   - [ ] 高考倒计时
   - [ ] 问候语
   - [ ] 一言小语
     - [ ] 换一句一言小语
   - [ ] 待办列表
 - [ ] 高级功能
   - [ ] 专注模式
     - [ ] 全屏化专注模式
     - [ ] 窗口化专注模式
   - [ ] 设置功能
     - [ ] 自定义字体
     - [ ] 自定义倒计时内容
     - [ ] 自定义主题

2. 按照提示安装
3. 安装完成
